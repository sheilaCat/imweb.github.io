  • 文本色：#333 $colorText
  • 背景色：#fff $colorBg
  • 边框色：#e5e5e5 $colorBorder

scss提供了%btn-primary,%btn-default,%btn-weak,%btn-m,%btn-s可以使用

表单

form表单的基础class为form,如果label与输入框同行则追加class form--h。如toC则追加class form--c(默认为toB侧的输入框高度为30px，而toC的为40px)

通过f-item上添加class表示状态,f-item--error（表示错误）,f-item--success（表示成功）,f-item--info（表示提示），如果一行有多个输入框，只表示一个输入框的错误状态可以在input上追加class text--error

radio与checkbox有三种特殊状态，分别为.checked(选中), .disabled(不可选), checked.disabled(选中不可更改)

select相关组件有两个状态，分别为.selecting(展开下拉框选择), .selected(已经选择了值)
